# EasyMark 

轻量级的 Markdown 在线文档

- 不赖数据库
- 无外部依赖，可内网部署
- 支持图片上传

## 部署说明
1. 需要使用nginx作为反向代理和文件服务器
2. 具体配置参考 /examples

## 引用
1. markdown编辑器：
[Milkdown](https://github.com/Saul-Mirone/milkdown)
2. ui: [naive-ui](https://github.com/tusen-ai/naive-ui)
3. icon: [xicons](https://github.com/07akioni/xicons)
